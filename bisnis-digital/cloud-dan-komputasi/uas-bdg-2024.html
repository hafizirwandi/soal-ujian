SOAL 1

Sebuah perusahaan rintisan (startup) di bidang logistik ingin beralih dari menyewa server fisik (on-premise) ke layanan Cloud Computing (seperti AWS atau Google Cloud). Pihak manajemen bingung tentang dampak finansial dari transisi tersebut.

Bandingkan keuntungan finansial dari pergeseran model pembiayaan CapEx (Belanja Modal di awal) menjadi OpEx (Biaya Operasional bulanan)! Analisislah mengapa model pembayaran "pay-as-you-go" pada Cloud dinilai lebih aman untuk menjaga arus kas (cash flow) sebuah startup yang pergerakan pasarnya masih fluktuatif.

SOAL 2

Sebuah klinik kesehatan swasta berencana untuk mendigitalkan seluruh rekam medis pasien mereka dengan menyimpannya di penyedia layanan Cloud eksternal komersial agar mudah diakses secara daring oleh dokter.

Sebagai analis bisnis digital, identifikasilah risiko-risiko bisnis dan hukum yang mungkin timbul jika suatu saat penyedia Cloud tersebut mengalami kebocoran data (data breach)! Rancanglah pedoman kebijakan internal (SOP Kepatuhan) secara non-teknis bagi staf klinik agar proses penyimpanan data ke Cloud tidak membahayakan privasi pasien.

SOAL 3

Sebuah perusahaan retail busana (fashion) tradisional ingin membuat toko online berskala nasional. Karena tidak memiliki tim IT internal, mereka memutuskan untuk menyewa layanan Cloud berbasis Software as a Service (SaaS) e-commerce siap pakai (seperti Shopify) dibandingkan membuat sistem dari nol.

Evaluasilah keputusan bisnis tersebut! Jelaskan kelebihan dan kelemahan strategis menggunakan layanan SaaS bagi perusahaan retail. Selain itu, bagaimana strategi bisnis yang harus disiapkan jika di masa depan mereka ingin menambahkan fitur khusus yang kebetulan tidak didukung/disediakan oleh platform SaaS tersebut?

SOAL 4

Banyak perusahaan tertarik menggunakan teknologi Cloud karena dijanjikan lebih hemat. Namun pada kenyataannya, banyak jajaran direktur yang terkejut karena tagihan Cloud bulanan mereka membengkak secara tidak terkendali (Cloud bill shock) karena setiap divisi bebas menyewa resource Cloud.

Sebagai manajer operasional digital, rumuskanlah tiga kebijakan tata kelola pembiayaan Cloud (Cloud Cost Management) yang ketat! Pastikan kebijakan Anda dapat mengatur agar setiap divisi tetap bisa berinovasi secara digital tanpa melebihi batas anggaran (budget) perusahaan.

SOAL 5

Sebuah rantai restoran cepat saji ingin mengimplementasikan "Dapur Pintar" menggunakan infrastruktur Cloud. Semua mesin kasir dan sensor suhu oven di seluruh cabang terhubung ke internet untuk mengirim data secara real-time ke kantor pusat.

Jelaskan nilai bisnis (business value) dari sentralisasi data tersebut terhadap peningkatan efisiensi bahan baku dan kualitas pelayanan pelanggan! Buatlah juga rencana mitigasi bisnis (contingency plan) di level operasional restoran jika tiba-tiba koneksi internet ke Cloud terputus total saat jam makan siang sedang padat!
